## вҡЎ What is VibeCheck?

> **VibeCheck** is a **multimodal, context-aware music streaming platform** that reads your **facial emotion**, checks the **weather outside**, and knows **what time of day** it is вҖ” then curates a 7-song playlist in real time. No typing. No searching. Just your face.

This project was built as a **Mini Project** for the paper:
> *"A Smart Multimodal Context-Aware Music Streaming Platform Using Facial Emotion Recognition"*

## рҹҡҖ Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/Chetan142006/VibeCheck-Emotion-Music-Player.git
cd VibeCheck-Emotion-Music-Player
```

### 2. Set up the environment
```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
# source .venv/bin/activate   # macOS/Linux

pip install -r requirements.txt
```

### 3. Configure API Keys
Create a `.env` file in the root directory:
```env
OPENWEATHER_API_KEY=your_openweathermap_key
SPOTIPY_CLIENT_ID=your_spotify_client_id
SPOTIPY_CLIENT_SECRET=your_spotify_client_secret
```

> рҹ”‘ Get your keys from:
> - [OpenWeatherMap](https://openweathermap.org/api) вҖ” Free tier works
> - [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)

### 4. Run it
```bash
python app.py
```
Open вҶ’ **http://localhost:5000**

## рҹ“Ў API Reference

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/detect-emotion` | Accepts base64 image вҶ’ returns emotion + confidence scores |
| `GET` | `/api/weather` | Fetches weather by lat/lon (falls back to IP geolocation) |
| `POST` | `/api/recommend` | Returns a 7-song playlist from the 3D context engine |
| `GET` | `/api/youtube-search` | 2-pass YT Music search with title validation |
| `POST` | `/api/like` | Save a liked song with its emotion context |
| `DELETE` | `/api/like` | Remove a liked song |
| `GET` | `/api/likes` | Fetch all liked song keys |

## рҹҺө Smart Playlist Logic

```
Priority 1 вҶ’ вқӨпёҸ  Liked songs matching current emotion
Priority 2 вҶ’ рҹҺ§  Spotify API recommendations (contextual)
Priority 3 вҶ’ рҹӣЎпёҸ  Curated safe fallbacks (always available)
```

- Tracks already played are stored in a **50-song rolling history** to avoid repetition
- Playlist auto-refreshes when the queue ends (**Infinite Queue mode**)
